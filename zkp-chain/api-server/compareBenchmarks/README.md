# compareBenchmarks

Benchmark suite for the Q1 journal paper comparing **Face** (Cosine Similarity) vs **Iris** (Hamming Distance) ZK-SNARK authentication circuits.

## Quick Start

```bash
# From the api-server/ directory

# Step 1: Run benchmarks (generates benchmark_results.json)
node compareBenchmarks/benchmark_paper.js --runs 5

# Step 2: Generate publication-quality figures
python3 compareBenchmarks/plot_results.py
```

## Benchmark Script

```bash
# Quick run (3 iterations)
node compareBenchmarks/benchmark_paper.js --runs 3

# Full paper-quality run (5+ iterations for statistical significance)
node compareBenchmarks/benchmark_paper.js --runs 5

# Also dump raw JSON to stdout
node compareBenchmarks/benchmark_paper.js --runs 5 --json

# Or via npm scripts
npm run benchmark:quick
npm run benchmark:paper
```

### Metrics Collected

| Category | Metrics |
|----------|---------|
| **1. Circuit Complexity** | R1CS constraints, variables, public/private inputs (from `.r1cs` files) |
| **2. Computational Latency** | Witness gen, proof gen, proof verify — median ± σ over N runs |
| **3. Storage & Payload** | `.zkey`, `.wasm`, `.vkey`, R1CS sizes, proof payload |
| **4. On-Chain Gas** | Groth16 verify gas, calldata, SSTORE, deploy (EIP-1108 estimates) |

## Plotting Script

```bash
# Default: reads benchmark_results.json, outputs PNG
python3 compareBenchmarks/plot_results.py

# High-quality PDF (better for papers)
python3 compareBenchmarks/plot_results.py --format pdf --dpi 600

# Also print LaTeX table markup
python3 compareBenchmarks/plot_results.py --latex

# Custom input/output
python3 compareBenchmarks/plot_results.py -i results.json -o ./figures/
```

### Figures Generated

| Figure | Description |
|--------|-------------|
| `figure1_constraints_vs_time` | R1CS Constraints & Proof Time — grouped bar (dual axis) |
| `figure2_storage_comparison` | Storage sizes — proving key, WASM, R1CS, vkey, proof |
| `figure3_latency_breakdown` | Computational latency — stacked bar (witness + proof + verify) |
| `figure4_gas_breakdown` | On-chain gas — stacked bar (verify + calldata + SSTORE) |
| `figure5_consistency` | Per-run scatter + box plot — timing consistency across runs |
